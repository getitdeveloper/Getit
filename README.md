# Getit

- it 프로젝트를 하고싶은 기획자, 디자이너, 개발자모두를 만나 모집할수 있는 플랫폼 입니다.
- 다른 플랫폼과는 다르게 현재 진행중인 프로젝트 전에 진행했던 프로젝트등 자신을 소개할수 있는 기능이 있습니다.
- 이외에도 직군별 질문/ 자유게시판을 마련하여 프로젝트 뿐만 아닌 서로의 생각,아이디어를 공유할수 있습니다.
